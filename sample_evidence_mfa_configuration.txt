## Configuration Details

### Key Settings Applied:

1. **MFA Enabled Globally:** All users required to enroll in MFA within 7 days of account creation

2. **Conditional Access Policies:**
   - Policy #1: Require MFA for all users (100% coverage)
   - Policy #2: Require MFA for admin roles (100% coverage)
   - Policy #3: Require MFA for risky sign-ins (risk-based)

3. **Supported Authentication Methods (in order of preference):**
   - Microsoft Authenticator (Push notification) - PREFERRED
   - Microsoft Authenticator (TOTP code) - PREFERRED
   - Hardware OATH tokens (YubiKey) - PREFERRED
   - Phone call - ACCEPTABLE
   - SMS code - ACCEPTABLE (Low-risk only)

4. **Session Management:**
   - MFA prompt frequency: Every 30 days for trusted devices
   - Immediate re-authentication required for: Location change, high-risk actions

5. **Monitoring & Alerting:**
   - Real-time alerts for MFA failures (3+ attempts)
   - Daily compliance dashboard sent to security team
   - Weekly executive summary report
